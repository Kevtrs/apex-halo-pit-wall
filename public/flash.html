Mode flash

Page de préparation pour flasher ton Guition. Le flash direct depuis navigateur marche seulement avec Chrome/Edge et Web Serial. Sinon utilise ESP Flash Download Tool ou PlatformIO.

1Branche le Guition en USBUtilise un câble USB data, pas juste recharge.
2Passe-le en mode boot si besoinMaintiens BOOT, appuie sur RESET, relâche RESET puis BOOT.
3Choisis le port sérieLe navigateur demandera COM3, COM4, USB Serial, CH340 ou CP210x.
4Flash le firmwareIl faudra ajouter les fichiers .bin quand le firmware LVGL sera prêt.
